REST API

Megam API Server is a stateless REST based cloud server. Our SDKs abstract away the REST interface and provide a set of handy methods you can interact with when deleting, creating, updating and deleting objects. We've also provided information on how we use HMAC to create authorization headers for use when accessing Megam.
